## Learning Path

```
🟢 Part 1: មូលដ្ឋាន (Beginner)
 │  HTML, CSS, JavaScript basics
 │  Mini Project: Personal Profile Website
 ▼
🟡 Part 2: Frontend Development
 │  ES6+, React, API Integration
 │  Mini Project: Interactive Web App
 ▼
🔵 Part 3: Backend Development
 │  Node.js, Express, MongoDB, Auth
 │  Mini Project: Complete REST API
 ▼
🟣 Part 4: Full-Stack Development
 │  Connect Frontend + Backend, CRUD, Roles
 │  Major Project: E-Commerce / Dashboard
 ▼
🟠 Part 5: Deployment & Real Project
 │  Deploy, Testing, Performance, Security
 │  Major Project: Live Deployment
 ▼
🔴 Part 6: Career Ready
    Portfolio, Collaboration, UX/UI, Job Prep
```

## Tools & Technologies

**Frontend:** HTML5, CSS3, Tailwind CSS, JavaScript (ES6+), React, Vite
**Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT
**Tools:** VS Code, Git, GitHub, Postman, Chrome DevTools
**Deployment:** Vercel, Netlify, Railway, MongoDB Atlas
**Design:** Figma (basics)
