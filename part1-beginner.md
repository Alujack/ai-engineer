## Lesson 1: កុំព្យូទ័រ & Internet ដំណើរការយ៉ាងដូចម្តេច

- យល់ដឹងពីរបៀបដំណើរការរបស់ Internet

- Internet & World Wide Web ខុសគ្នាយ៉ាងដូចម្តេច
- Client vs Server
- HTTP / HTTPS Protocol

## Lesson 6: CSS Layout

### មាតិកា (Topics)
- **Flexbox**
  - `display: flex`
  - `flex-direction`, `justify-content`, `align-items`
  - `flex-wrap`, `gap`

- **CSS Grid**
  - `display: grid`

  - `grid-template-areas`
- **Positioning**
  - `static`, `relative`, `absolute`, `fixed`, `sticky`
  - `z-index`
- **Overflow** (`hidden`, `scroll`, `auto`)

## Lesson 7: Responsive Design

### គោលបំណង (Objectives)
- អាចបង្កើត Website ដែលស្អាតលើគ្រប់ Screen Size
- យល់ដឹងពី Mobile-First Approach

### មាតិកា (Topics)
- Viewport Meta Tag (`<meta name="viewport">`)
- Media Queries (`@media`)
  - Breakpoints: Mobile (< 768px), Tablet (768px - 1024px), Desktop (> 1024px)
- Mobile-First vs Desktop-First Approach
- Responsive Images (`max-width: 100%`, `<picture>`, `srcset`)
- Responsive Typography (clamp, vw units)
- Responsive Navigation (Hamburger Menu concept)
- CSS Variables (`--custom-property`)
- Transitions & Animations មូលដ្ឋាន

### លំហាត់ (Exercise)
- ធ្វើឱ្យ Layout ពី Lesson 6 ក្លាយជា Responsive
- បង្កើត Landing Page ដែល Responsive (Mobile + Desktop)
